# Vrinda Store Sales Data Analysis using Excel

## Project Overview

This project analyzes sales performance using Excel based on the Superstore Sales Dataset.

The goal is to discover sales trends, profitable categories, and regional performance through data analysis and visualization.

## Dataset

Source: Kaggle – Superstore Sales Dataset

Important Columns Used:

- Order Date
- Region
- Category
- Sub-Category
- Sales
- Profit
- Customer Name

## Business Questions Solved

1. Which region generates the highest sales?
2. Which category sells the most?
3. What is the monthly sales trend?
4. Which product/sub-category is most profitable?

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design
- Data Visualization

## Project Workflow

1. Dataset collection from Kaggle
2. Data cleaning
3. Data preprocessing
4. Pivot table analysis
5. Dashboard creation
6. Business insight generation

## Key Insights

- Highest revenue generating region identified.
- Top selling category discovered.
- Monthly sales patterns analyzed.
- Most profitable product category determined.
